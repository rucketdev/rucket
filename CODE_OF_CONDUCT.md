# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Summary

We are committed to providing a welcoming and inclusive environment for everyone.

## Reporting

If you experience or witness unacceptable behavior, please report it by contacting the project maintainers.
