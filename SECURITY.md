# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting Issues

Please use GitHub's private security advisory feature to report issues:
https://github.com/rucketdev/rucket/security/advisories/new

Do not report via public GitHub issues.

## Response Timeline

- Acknowledgment: Within 48 hours
- Initial assessment: Within 7 days

## Configuration

- Store configuration files with restricted permissions
- Use strong credentials in production
- Consider deploying behind a reverse proxy with TLS
